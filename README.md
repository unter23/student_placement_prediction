🎓 Placement Predictor
A machine learning project to predict whether a student will be placed based on academic and practical factors.

Features
Predict placement using ML model
Analyze importance of CGPA, internships, etc.
Interactive web app using Streamlit
Tech Stack
Python
Pandas, NumPy
Scikit-learn
Streamlit
Run Project
Install dependencies: pip install -r requirements.txt

Train model: python train.py

Run app: streamlit run app.py
